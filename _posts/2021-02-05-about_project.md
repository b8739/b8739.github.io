# 프로젝트 참여 배경
현재 내가 인턴으로 소속된 R&D 회사에서 머신러닝 웹 개발을 맡게 되었다.  

설립된 지 2-3년 밖에 안 된 스타트업 규모의 회사이고, 오로지 Data Science 인력으로만 이루어진 회사이기 때문에, 사수나 동기 없이 대표님과 1대1로 커뮤니케이션을 해가면서 프로젝트를 진행하고 있다.  

개발에 필요한 내용을 독학하는 것에 있어서 약간의 어려운 부분이 있지만 오히려 폭 넓은 경험을 할 수 있을거라고 기대하고 있다.  

나의 개인 프로젝트가 아니기 때문에 구체적인 정보를 공개할 수는 없지만, 진행하면서 해결해야 하는 문제나 관련 공부에 대해서 포스팅하려고 한다.  

---
# 프로젝트 목표
개발하려는 웹 어플리케이션은  

1) data를 import 받아서 
2) preprocess
3) modeling 한 후 
4) 사용자에게 다양한 그래프로 시각화해서 보여주는 웹 개발을 목표로 하고 있다.    

쉽게 말해서 일반적인 머신러닝 웹이다. 인턴 기간이 많이 남지는 않았기 때문에 기본적인 기능 구현이 가능한 prototype 형태의 웹 개발을 목표로 하고 있다.

# 관련 프로그래밍 언어/지식
1. Flask (벡엔드 서버를 위한 프레임워크로 사용)
2. HTML, CSS, Javascript
3. React (어쩌면 Angular, Vue) 
4. Liquid
